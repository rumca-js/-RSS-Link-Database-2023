  <p id="NG42kD">Microsoft is no longer updating WordPad and plans to remove the word processor from a future release of Windows. The software giant will instead recommend Microsoft Word, its paid word processor that has always been far more feature rich than the basic WordPad app that has shipped as part of Windows since Windows 95.</p>
