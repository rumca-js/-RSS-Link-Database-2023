Invader Studios is hard at work on the upcoming Daymare 1994: Sandcastle, which is a prequel to the 2019 horror game Daymare 1998: Sandcastle. Set 4 years before the event of the original game, players will be assuming the role of Dalila Reyes as she investigates a secret government facility crawling with all kinds of undead threats. 

The game is heavily inspired by Dead Space, and has plenty of promising tricks that has us excited for this release. While there’s no doubt that there’s plenty to be gained for a game by having more open design that allows for a greater emphasis on exploration, more optional content, and, in turn, more play time, there’s also no doubting the value of a hyper-linear experience that places almost complete emphasis on forward momentum, to the exclusion of almost everything else. 

From heart-pounding action to great storytelling to memorable set pieces and so much more, a game that chooses to walk down that road can, if handled by a talented developer, end up boasting many (or all) such strengths, making for experiences that can stand as some of the medium’s best offerings.
