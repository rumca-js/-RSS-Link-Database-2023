## Apple Vision Pro Is Impressive, But I’m Worried…
 - [https://www.youtube.com/watch?v=uUblFIaZKIk](https://www.youtube.com/watch?v=uUblFIaZKIk)
 - RSS feed: https://www.youtube.com/feeds/videos.xml?channel_id=UCO2x-p9gg9TLKneXlibGR7w

Apple’s Vision Pro headset has amazing hardware that’ll be best-in-class. Perhaps best, period. But here’s the thing… hardware hasn’t been the problem with MR/VR in years. I’m not sure Apple’s new headset targets the true issue with head mounted displays: compelling software that complements a device that requires intentionality.
